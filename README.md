# SkillFactoryQAP-23.2
/SkillFactory homework repository for 23.2.1(PJ)/

Задача кода:
Построить парсер, принимающий год выхода фильма, возвращающий информацию об оценённых фильмах за этот год с сайта www.kinoafisha.info
Сбор этой информации открывает возможность не только собрать список фильмов, вышедших за определённый год, но также и произвести сравнение по выставленной оценке с другими сервисами.
Также, при необходимости, код можно расширить, включив в выдачу параметры жанра и/или страны-производителя. Тогда станет доступно сравнение жанра или страны с итоговой оценкой,
а также последующий анализ данных в зависимости от года выхода. Например анализ популярности того или иного жанра из года в год, и возможное влияние мировых событий на эту статистику.

Сборка и запуск:
Для работы кода необходимы библиотеки requests и bs4, BeautifulSoup, также для вывода данных (в формате xlsx) необходим pandas.
Вся работа происходит в рамках функции collect_films_per_year, которую можно скопировать и интегрировать в основной код.
Принимается 1 аргумент integer, год выхода, возвращается список list с объектами в виде словарей dict {'film_name': film_name, 'rating': rating}.
После функции написан код, с помощью которого можно произвести проверку работы функции, и также преобразование выходящих данных в файл-таблицу excel.

В репозитории с этим кодом приложены примеры работы кода в формате таблиц excel.
